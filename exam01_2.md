# 迴圈 #

## 說明 ##

請使用者輸入數字，直到-1停止<br>
並算出:<br>
1. 數字的加總<br>
2. 數字的平均<br>
3. 共輸入的幾組數字<br>

*若使用者輸入非數字之字串，程式不會出錯。
## Input Format ##

數字(int/float)或文字(str) <br>
-1<br>

## Output Format ##

1. 數字的加總(float至小數點第二位)<br>
2. 數字的平均(float至小數點第二位)<br>
3. 共輸入的幾組數字(int)<br>

## Sample Input 1 ##
```
82<br>
54.3<br>
74<br>
-1
```

## Sample Output 1 ##
```
210.30<br>
70.10<br>
3
```

## Sample Input 2 ##
```
55<br>
54.5<br>
82<br>
NA<br>
23<br>
-1
```

## Sample Output 2 ##
```
214.50<br>
71.50<br>
4
```
## Hint ##

```
None
```
